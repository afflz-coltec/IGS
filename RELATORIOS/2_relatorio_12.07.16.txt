O que fizemos?
Hoje decidimos que vamos usar o Raspberry Pi para controlar a câmera e receber sinais do Arduino nano e informação do jardim via rede. Além disso, vamos usar um display Nokia para mostrar as informações do jardim.


Quais foram as dificuldades encontradas?
Tivemos dificuldades em escolher entre o Raspberry Pi e o Arduino para esse trabalho, pois vamos controlar vários sinais analógicos e não temos um conversor A/D para fazer este trabalho pelo Raspberry Pi. Além disso tivemos problemas para usar o display TFT por questões de compatibilidade com o Raspberry.

O que será feito para superar a(s) dificuldades semanais?
Para superar essas dificuldades vamos usar o Arduino para converter os sinais analógicos e enviá-los para o Raspberry Pi e o display Nokia ao invés do TFT pela falta de uma interface que nos possibilitasse o uso do TFT.
